카드 목록

    버튼/폼/색상 토큰을 활용한 일관성 있는 UI설계.

    온보딩부터 결제까지 이탈을 줄이는 플로우 설계.

  • Component

    토큰 드리븐 스타일과 컴포넌트 분해 전략.

  • Typography

    계층적 스케일과 리듬, 가독성 개선.

  • Information Architecture

    내비게이션과 검색성을 높이는 구조 설계.

  • Build

    ESLint/Prettier, 번들링, 최적화 베이직.

  • States

    Success/Warning/Error 토큰화.

  • Usability

    테스트 시나리오와 정량 지표 설계.

  • A11y Audit

    명암비/키보드 탐색/aria 속성 점검 방법.
